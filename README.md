# AI Trading Platform

Monorepo for AI Trading platform built with React TS + NestJS.

## Structure
```
apps/
  web/     — React 18 + TypeScript + Vite + Tailwind CSS
  api/     — NestJS + JWT Auth
packages/
  shared/  — Shared TypeScript types
```

## Quick Start
```bash
npm install
npm run dev:web   # Frontend on :5173
npm run dev:api   # Backend on :3000
```
